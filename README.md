# Explainable PrivacyAssist

Lokaler React/TanStack-Start-Prototyp fuer eine erklaerbare Privacy-Assist-Demo.

## Start

```bash
npm install
npm run dev
```

Danach die von Vite ausgegebene lokale URL oeffnen.

## Verfuegbare Befehle

```bash
npm run build
npm run lint
npm run preview
```

## Routen

- `/`
- `/demo`
- `/categories`
- `/script/$id`
- `/compare`
- `/about`

Die App nutzt Mock-Daten aus `src/data/mockSites.ts`; echte Browser-Extension- oder Tracking-Erkennung ist bewusst nicht enthalten.
